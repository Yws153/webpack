## 3.换行

## 4.空行
* 一段代码的语义和另一段代码不相关，用空格隔开
```
if () {
	
	for () {

		if () {

		}

	}
}
```

* 函数：驼峰大小写法（以动词开头）
* 常量：大写和下划线来命名

## 8.对象直接量

```

book.title = 'xxx'
book.author = 'xxx'
```

```

	title: 'xxx'
	author: 'xxx'
}
```

## 9.数组直接量


## 10.注释
* 勤写代码
* 注释的前面加空行
* 对于可能被误认为错误的地方一定写注释

## 11.if
```
if (let i = 0; i > 10; i++) {
	doSomeThing()
}
doSomeOtherThing()

if () {
	doSomeThing()
} else {
	doSomeOtherThing()
}
```
